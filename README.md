
# 🍕 Pizza Sales Dashboard

## Project Overview
This project presents an interactive Power BI dashboard for analyzing pizza sales data.

## Tools Used
- Power BI
- Excel
- DAX

## Dashboard Features
- Total Revenue
- Total Orders
- Total Pizzas Sold
- Average Order Value
- Sales by Category
- Best & Worst Selling Pizzas
